# Locafy (demo web)

Trang web demo cho app "tìm trọ quanh trường đại học".

## Cách chạy

Vì không có backend, bạn chỉ cần mở trực tiếp các file:

- `index.html` (trang chủ)
- `list-room.html` (trang đăng phòng)

Tin đăng được lưu vào trình duyệt bằng `localStorage` để bạn xem trước bộ lọc.
