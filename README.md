# Locafy (demo web)

Trang web demo cho website "tìm và đăng trọ uy tín" cho sinh viên và chủ trọ.

## Cách chạy

Vì không có backend, bạn chỉ cần mở trực tiếp các file:

- `index.html` (trang chủ)
- `list-room.html` (trang đăng phòng)

Tin đăng được lưu vào trình duyệt bằng `localStorage` để bạn xem trước bộ lọc.

## Ghi chú

Khi bạn muốn làm bản thật, có thể thay phần `localStorage` bằng API/Backend (Node/Python/Go...).
